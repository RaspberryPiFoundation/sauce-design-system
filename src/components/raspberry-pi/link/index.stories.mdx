# Link

If you can influence the HTML, use this HTML class to style links. If you'd like to style links that have been defined in a CMS, you may instead use the `Publication` component.

<Story name="Vanilla">
  <div style={{ padding: '20px' }}>
    <a class="sc-rp-link" href="#">
      Link text goes here
    </a>
  </div>
</Story>

## With Inverted Text Modifier

We can specify an additional `sc-rp-link--inverted-text` modifier to swap the text color:

<Story name="With Inverted Text Modifier">
  <div style={{ backgroundColor: '#002B3F', padding: '20px' }}>
    <a class="sc-rp-link sc-rp-link--inverted-text" href="#">
      Link text goes here
    </a>
  </div>
</Story>

## With Inverted Text Sauce Utility

We can also specify that a link should use inverted text styles on a higher component that has the `data-sauce-inverted-text` data attribute:

<Story name="With Inverted Text Sauce Utility">
  <div
    data-sauce-inverted-text
    style={{ backgroundColor: '#002B3F', padding: '20px' }}
  >
    <a class="sc-rp-link" href="#">
      Link text goes here
    </a>
  </div>
</Story>

We can also apply the same utility to the link itself.
