# Typography

## Fonts

Include this in your application `<head>`. The following example includes expanded whitespace so you can see what it is, but feel free to minify it in your application.

```html
<link
  crossorigin
  href="https://fonts.gstatic.com"
  rel="preconnect"
/>
<link
  as="style"
  href="https://fonts.googleapis.com/css?family=Rubik:300,400,500,700|Space+Mono&display=swap"
  rel="preload"
/>
<link
  href="https://fonts.googleapis.com/css?family=Rubik:300,400,500,700|Space+Mono&display=swap"
  media="print"
  onload="this.media='all'"
  rel="stylesheet"
/>
<noscript>
    <link
      href="https://fonts.googleapis.com/css?family=Rubik:300,400,500,700|Space+Mono&display=swap"
      rel="stylesheet"
    />
</noscript>
```

This HTML fragment makes use of Resource hinting to provide the fastest download of the typefaces we use. It looks counter-intuitive to include more HTML to achieve this aim, but it helps the browser correctly prioritise and handle the request.

For a deep-dive explanation of what's happening, read Harry Roberts' guide here: https://csswizardry.com/2020/05/the-fastest-google-fonts/

## Font Family

Name          | Value
--------------|-----------------------------------------------------
`$serif`      | 'Rubik', serif;
`$sans-serif` | 'Rubik', sans-serif;
`$monospace`  | 'Space Mono', 'Lucida Console', 'Monaco', monospace;

```scss
@use 'src/tokens/font-family';

.some-selector {
  font-family: font-family.$monospace;
}
```

## Typographic Scale

Name           | 0–800px | 800–1200px | 1200–∞ | Scale
---------------|---------|------------|--------|-----------
`$rollerskate` | 12.8    | 16         | 19.2   | &times;0.8
`$skateboard`  | 16      | 20         | 24     | &times;1
`$scooter`     | 19.2    | 24         | 28.8   | &times;1.2
`$bike`        | 24      | 30         | 36     | &times;1.5
`$moped`       | 32      | 40         | 48     | &times;2
`$car`         | 40      | 50         | 60     | &times;2.5
`$van`         | 56      | 70         | 84     | &times;3.5

```scss
@use 'src/mixins/type-scale';

.some-selector {
  @include type-scale('bike');
}
```

## Font Weight

Name       | Value
-----------|------
`$light`   | 300
`$regular` | 400
`$medium`  | 500
`$bold`    | 700

```scss
@use 'src/tokens/font-weight';

.some-selector {
  font-weight: font-weight.$light;
}
```

## Line Heights

Name       | Value
-----------|------
`$loose`   | 1.6
`$regular` | 1.4
`$cosy`    | 1.2
`$tight`   | 1.1

Use like this:

```scss
@use 'src/tokens/line-height';

.some-selector {
  line-height: line-height.$cosy;
}
```
