# Spacing

Use the `padding` and `margin` modules to apply spacing to your elements.

Try to avoid using bare `padding` and `margin` rules, so that when the time comes it will be easier to update designs to use [Logical Properties](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Logical_Properties/Basic_concepts), which is a more modern way of achieving these results in a multi-lingual web with modern layout tools such as Flexbox and Grid.

## Margin

```scss
@use 'src/mixins/margin';

.some-selector {
  // All four sides
  @include margin.all('ant');

  // Vertical axis
  @include margin.v('bee');

  // Horizontal axis
  @include margin.h('chaffinch');

  // Top
  @include margin.t('dog');

  // Right
  @include margin.r('eagle');

  // Bottom
  @include margin.b('flamingo');

  // Left
  @include margin.l('grizzly');
}
```

## Padding

```scss
@use 'src/mixins/padding';

.some-selector {
  // All four sides
  @include padding.all('ant');

  // Vertical axis
  @include padding.v('bee');

  // Horizontal axis
  @include padding.h('chaffinch');

  // Top
  @include padding.t('dog');

  // Right
  @include padding.r('eagle');

  // Bottom
  @include padding.b('flamingo');

  // Left
  @include padding.l('grizzly');
}
```
